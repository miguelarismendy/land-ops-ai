# prompts

System prompts for each AI agent in the land-ops-ai system.

Each file here is the personality, rules, and instructions given to a specific AI agent.
Think of these as the job descriptions for each AI worker.

## Files

- `morning-briefer.md` — How the morning agent talks to the owner, what it asks, how it waits
- `lead-qualifier.md` — Criteria for scoring a land lead (motivated seller signals, price range, location filters)
- `sms-scheduler.md` — How to build SMS campaigns, tone, timing rules, confirmation logic
- `follow-up-agent.md` — Follow-up cadence rules for sellers vs buyers, when to stop, what to say
- `offer-agent.md` — How to calculate and send blind offers on land parcels
- `voice-agent.md` — How to conduct calls, negotiate, handle objections, when to escalate to owner
